discoBit

discoBit is a configuration repository which enables remote configuration of applications. Configurations are read from the central server during bootstrap or on demand.

Hybris Agent

Your Hybris instances contact the discoBit repository via REST and read the applicable properties while bootstrapping. Apply the Java Agent to the Tomcat wrapper configuration together with the unique identifier for the calling node and security credentials to access the remote configuration. The properties are injected to the system context.
